# Maduinos

FPGA education, board bring-up, and embedded hardware prototyping examples.

Maduinos publishes compact public examples for FPGA learning and hardware prototyping. The main business focus is FPGA education, Verilog/SystemVerilog examples, board-level bring-up, and small consulting projects.

## Main Focus

- FPGA training and example design
- C_DUINO_A7 board demos
- Verilog/SystemVerilog learning material
- Embedded hardware prototyping
- Practical bring-up notes for learners and customers
