# Maduinos

FPGA 교육, 보드 bring-up, 임베디드 하드웨어 실습 예제를 정리하는 공개 저장소입니다.

Maduinos는 FPGA 학습자와 하드웨어 프로토타입 제작자를 위한 실습형 예제를 공개합니다. 주요 관심사는 Verilog/SystemVerilog 교육 자료, C_DUINO_A7 보드 예제, 보드 레벨 bring-up, 그리고 실제 프로젝트에 바로 참고할 수 있는 작은 설계 예제입니다.

## 주요 분야

- FPGA 교육 및 예제 설계
- C_DUINO_A7 보드 데모
- Verilog/SystemVerilog 학습 자료
- 임베디드 하드웨어 프로토타이핑
- 학습자와 고객을 위한 실무형 bring-up 기록

## 개인 실험 저장소

아래 저장소는 공개 개인 실험/취미 프로젝트입니다. FPGA 교육 및 비즈니스 포트폴리오와는 구분해서 관리합니다.

| 저장소 | 설명 |
| --- | --- |
| [`SlimePet`](https://github.com/maduinos/SlimePet) | 데스크톱 장난감 앱 실험 |
| [`macroKey`](https://github.com/maduinos/macroKey) | Arduino HID 키패드와 Windows용 Python 매크로 GUI 앱 |
| [`turntable`](https://github.com/maduinos/turntable) | Arduino 스테퍼 모터 턴테이블 실험 |
| [`Burndori`](https://github.com/maduinos/Burndori) | OCR 기반 영어 자막 인식 및 한국어 번역 오버레이 실험 |
| [`GIFtoRGB565`](https://github.com/maduinos/GIFtoRGB565) | GIF 프레임을 Arduino용 RGB565 `PROGMEM` 헤더로 변환하는 GUI 도구 |
| [`KiCAD_LIB`](https://github.com/maduinos/KiCAD_LIB) | KiCad 심볼, 풋프린트, 3D 모델을 모아둔 개인 라이브러리 |
