운영 흐름

저장소 작업은 로컬 점검, 저장소별 커밋, 직접 푸시, 공개 문서 관리로 나누어 처리합니다. 이 프로필 저장소는 전체 작업 폴더를 어떻게 다룰지 기록하는 기준 문서 역할을 합니다.

1. 로컬 점검

작업 폴더에서 변경된 저장소의 소스와 문서를 확인하고, 사용 가능한 도구로 문법 오류를 점검합니다.

2. 저장소별 커밋

상위 작업 폴더는 Git 저장소가 아니므로 변경된 하위 저장소 안에서 각각 커밋합니다.

3. 직접 푸시

각 저장소의 origin/main으로 푸시합니다. 여러 저장소를 다룰 때도 저장소별 상태를 먼저 확인합니다.

4. 범위 구분

FPGA 교육과 하드웨어 bring-up을 먼저 보여주고, 비-FPGA 저장소는 개인 실험으로 표시합니다.
