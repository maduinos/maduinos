# Release Process

This profile repository is published to the special GitHub profile repository `maduinos/maduinos`.

## Before Release

1. Run `./scripts/status-all.sh`.
2. Confirm `README.md` keeps FPGA and C_DUINO_A7 as the main public signal.
3. Update `CHANGELOG.md` when profile or operations guidance changes.
4. Push after GitHub authentication and DNS access are working.
